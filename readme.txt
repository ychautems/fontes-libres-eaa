Alors premièrement, pour faire des choix, nous alors déposer tous nos fichiers (fonts, spécimens, textes,) dans le dossier sur le serveur: seminaire_open_type  ->  propositions. 
Pour les fonts, je vous demanderais de déposer le dossier complet avec la license.txt (j'ai mis un exemple correct avec la font roboto).
Pour les spécimens, veuillez déposer un pdf de chaque tests que vous avez avec votre nom (par exemple: yannick01.pdf).
Pour les textes, un text au format RTF ouvrable avec TextEdit va très bien. Nommez les avez le nom titre du texte (par exemple: blanc_et_noir.rtf).

Merci de bien suivre ces instructions pour que l'on ne s'emmêle pas les pinceaux car il y aura vite beaucoup de fichiers.
